The DND Numbers module allows you to manage phone numbers that should not be contacted through certain communication channels. DND stands for **Do Not Disturb** ‚Äî these are numbers where people have requested not to receive specific types of messages or calls. When you add a number to this list, the system remembers which channels to block, helping you respect preferences and stay compliant.

<Callout type="info" emoji="‚ÑπÔ∏è">
  **What are DND Numbers?**
  
  Numbers on this list have one or more communication channels blocked (email, SMS, calls, voicemail, or inbound calls/SMS). The system uses this list to prevent sending messages or placing calls through blocked channels.
</Callout>

The DND Numbers module allows you to view, add, edit, and delete do-not-contact entries. You can filter by date range and DND status, and search the list by name, email, area code, or who added the entry.

## üöÄ Accessing the DND Numbers Page

1. Navigate to the **DND Numbers** section in your main menu
2. Click on **DND Numbers** from the options
3. The DND Numbers page will load showing all entries (default: numbers added today)

## üö¶ Understanding DND Status

The **DND Status** column shows colored badges indicating which communication channels are blocked for each number. Each badge type has a distinct color; a single number can have **multiple badges** if multiple channels are blocked (e.g. both DND Email and DND SMS).

| Badge | Color | Meaning |
|-------|--------|---------|
| **DND All** | Red | Do not contact through any channel ‚Äî no email, SMS, calls, or voicemail |
| **DND Email** | Blue | Do not send emails |
| **DND SMS** | Green | Do not send text messages |
| **DND Call Voicemail** | Yellow | Do not place calls or leave voicemails |
| **DND Inbound Call SMS** | Light blue | Do not allow incoming calls and SMS |

## ‚ûï Adding a New DND Number

**Purpose:** Add a phone number to the DND list and specify which communication channels to block.

**How it works:**

1. Click the **"Add DND Number"** button at the top of the page
2. Fill in the form (phone number is required; name, area code, and email are optional)
3. Select which channels to block under **DND Status**
4. Click **"Submit"** to save; the number appears in the list and a success message is shown

<img src="/images/DND-number/add-form.png" alt="Add DND Number Form" className="rounded-lg border shadow-sm mb-4" />

**Required field:** **Phone Number** ‚Äî Enter in any format; the system will format it correctly.

**Optional fields:** **First Name** (up to 20 characters), **Last Name** (up to 20 characters), **Area Code** (up to 5 characters), **Email** (up to 150 characters; must be valid).

**DND Status ‚Äî choose which channels to block:**

- **DND all channels** ‚Äî Blocks email, SMS, calls, and voicemail. When selected, you cannot select individual channels below.
- **Emails** ‚Äî Block email to this number
- **Text Messages** ‚Äî Block SMS to this number
- **Calls & Voicemails** ‚Äî Block calls and voicemail to this number
- **DND Inbound Calls and SMS** ‚Äî Block incoming calls and text messages (can be used alone or with others)

<Callout type="warning" emoji="‚ö†Ô∏è">
  **Important:** If you select **DND all channels**, the individual channel options are disabled. If you select any individual channel, **DND all channels** is automatically unchecked.
</Callout>

## ‚úèÔ∏è Editing an Existing DND Number

**Purpose:** Update contact information or change which channels are blocked for a number already on the list.

**How it works:**

1. Find the number in the table and click **"Edit"** in the **Action** column
2. The form opens with the current information; update any fields (first name, last name, area code, email, phone number, or DND Status)
3. **DND Status** checkbox changes are saved **immediately** when you change them ‚Äî you do not need to click Submit for those
4. Click **"Submit"** to save all other changes; the form closes and the list updates with a success message

<img src="/images/DND-number/edit-form.png" alt="Edit DND Number Form" className="rounded-lg border shadow-sm mb-4" />

**Note:** When editing, DND Status checkbox changes are saved right away. You can continue editing other fields and then click Submit to save the rest.

---

## üóëÔ∏è Deleting a DND Number

**Purpose:** Remove a number from the DND list permanently.

**How it works:**

1. Find the number in the table and click the **delete (trash) icon** in the **Action** column
2. A confirmation dialog appears asking **"Are you sure?"**
3. Click **OK** to confirm deletion or **Cancel** to keep the number
4. If you confirm, the number is removed from the list and a success message is shown

**Warning:** Deletion is permanent. To have the number on the list again, you must add it as a new entry.

## ‚öôÔ∏è How the Module Works

**When you add a number:** The system stores all information you provide and your DND Status selections, records who added the number and when, and which company it belongs to. The number is immediately active and used to block the selected channels.

**When you edit:** You can change any contact or DND fields. DND Status checkbox changes are saved immediately; clicking **Submit** saves the rest (name, phone, email, area code, etc.).

**DND status logic:** **DND all channels** is the most restrictive and clears individual channel options. Individual options (Emails, Text Messages, Calls & Voicemails) can be combined. **DND Inbound Calls and SMS** is independent and can be combined with others or used alone. Selecting **DND all channels** clears individual channels; selecting an individual channel clears **DND all channels**.

**Phone number formatting:** You can enter the number in any format (with or without dashes, spaces, or parentheses); the system converts it automatically. **Time zone:** All dates and times are shown in your account's time zone. **Company restrictions:** Non-administrators see only their company's DND numbers; administrators see all companies. The system tracks company per number automatically.
