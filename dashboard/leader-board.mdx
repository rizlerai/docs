# Lead Board

The Lead Board provides comprehensive lead tracking and performance analysis with advanced filtering options and export capabilities.

## Interface Overview

The Lead Board interface features a sophisticated filtering system and multiple view options for analyzing lead performance and agent productivity.

## Filter Controls

### Date Range Selection
- **Date Range Dropdown** - Quick selection for time periods
  - "Today" - Current day view (default selection)
  - Custom date range options available
- **Start Date Field** - Specific start date picker (format: MM/DD/YYYY)
  - Current: 07/07/2025
- **End Date Field** - Specific end date picker (format: MM/DD/YYYY)  
  - Current: 07/07/2025

### Advanced Filtering Options

#### Filter By Section
- **Filter By Dropdown** - "Disposition" filtering category
  - Allows filtering by lead disposition status
  - Customizable filter categories
- **Agents Dropdown** - Agent selection filter
  - "All" - Shows data for all agents (default)
  - Individual agent selection available
- **Disposition Field** - Specific disposition filter
  - Text input: "dev" (current filter)
  - Custom disposition status filtering

### View Configuration

#### View By Section
- **Board View** - Current selected view mode
  - Visual board layout for lead tracking
  - Card-based display interface
  - Alternative view modes available

## Action Controls

### Primary Actions
- **Filter Button** - Apply selected filter criteria
  - Blue action button for executing filters
  - Real-time data filtering
- **Export Button** - Data export functionality
  - Blue action button for data export
  - Multiple export format options
