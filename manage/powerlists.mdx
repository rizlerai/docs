### 📈 PowerList Fields

| Field | Description | Example |
|-------|-------------|---------|
| **#ID** | PowerList identifier | `1`, `2`, `3`, `4`, `5`, `6` |
| **Active** | List status indicator | Active/Inactive toggle |
| **ID** | Unique list code | `MjcxNT14`, `MjcxNTIy`, `MjcxMzQw` |
| **PowerList** | List name/description | `Dev Test`, `Excel File Upload Test`, `Testinggg` |
| **# Uploaded** | Total contacts uploaded | `3595`, `784`, `1455`, `0`, `3`, `43` |
| **# Dialed** | Contacts called | `0`, `0`, `0`, `0`, `1`, `2` |
| **# Daily Dialed** | Daily call volume | `0`, `0`, `0`, `0`, `2`, `2` |
| **# Removed** | Contacts removed | `0`, `0`, `0`, `0`, `0`, `0` |
| **Created At** | Creation timestamp | `08-04-2025 10:57 AM` |
