### üß† 2. Key Terms & Concepts

| **Term**           | **Definition**                                                                                          |
|--------------------|--------------------------------------------------------------------------------------------------------|
| **PowerList**      | Smart container for segmenting contacts, fully tweakable for campaign success & flexible organization   |
| **Disposition**    | Call outcome label triggering actions (e.g., "Success", "Remove", "Post to CRM")                       |
| **Priority**       | Determines list order in dial queue ‚Äî put your hottest leads first!                                    |
| **Max Contacts**   | Limit of contacts per list (minimum: 50 for max power)                                                 |
| **Dial at a Time** | Outbound call concurrency ‚Äî how many calls hit the lines together                                      |
| **Call Timeout**   | How long the dialer waits before auto-hangup on no answer                                              |
| **Cool Down Time** | Wait time before retrying a failed contact (keep it legal & people-friendly)                           |
| **GHL**            | GoHighLevel CRM ‚Äî sync leads instantly, automate follow-up, and turbocharge campaigns                  |
| **DNC**            | Do Not Call ‚Äî stays compliant by never dialing protected numbers                                        |
| **Contact Fields** | Your custom data points (name, phone, tags, notes, etc.)                                               |
| **Tags**           | Flexible labels for searching, filtering, & organizing like a pro                                      |
| **AI Agent**       | Virtual assistant that can handle callbacks & follow-ups                                               |
| **Default Settings** | One-click re-use of your favorite list "recipes"                                                     |
| **Walkthrough**    | Interactive, in-app guide to make PowerLists setup effortless                                          |

- All PowerList names must be unique‚Äîno duplicates!
- Minimum list size: 50 contacts (for dialer optimization).
- File formats: `.csv` and `.xlsx` supported.
- Monitor large uploads for errors or incomplete processing.
- Maintain data privacy, accuracy, and follow access rules.

### üõ†Ô∏è PowerList Edit Interface Breakdown

|-----------------------------------|-----------------------------------------------------------------------------------------------|

| **Custom Fields/Dispositions**    | Create custom fields and dispositions with specified actions/results (e.g., "No Action")        |
