### üõ†Ô∏è PowerList Edit Interface Breakdown

| Section                          | What You Can Edit/Configure                                                                     |
|-----------------------------------|-----------------------------------------------------------------------------------------------|
| **PowerList Member & Name**       | View/update the PowerList's display name and assigned member                                   |
| **Max Contacts & Disposition**    | Limit the number of contacts in the list and set max allowed disposition (call) attempts       |
| **Dialing Controls**              | Set dial-at-a-time, max attempts per contact (daily/total), and customize dial timeout          |
| **Priority & Automation**         | Adjust priority, set next PowerList logic, choose to ignore DNC (Do-Not-Call) as needed        |
| **Tags & Agent Controls**         | Assign tags, select callback agents, enable/disable reset for dialed contacts, and set cooldown|
| **GHL Integration**               | Pick "GHL Location", set "GHL Tag on Disconnected" for tight external platform integrations     |
| **Custom Fields/Dispositions**    | Create custom fields and dispositions with specified actions/results (e.g., "No Action")        |
| **Upload & Import Data**          | Drag-and-drop CSV/Excel to add records; use mapping tool to align import fields with PowerList  |

---

### ‚úèÔ∏è How to Edit a PowerList

1. Click on the **More Actions (‚ãÆ)** beside your PowerList.
2. Choose **Edit**.
3. Update any field/setting in the Edit interface.
4. Click **Save** to apply your changes.
