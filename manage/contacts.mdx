**Contacts** is where you view and work with people stored in your **PowerLists**. You can choose a list, search and sort rows, show/hide columns, open a person’s conversation, delete contacts, and run bulk actions (move list, update tags, delete).

## 📂 Choosing a PowerList

Use **Select PowerList** to choose which list fills the table.

- If you arrived from another page that already selected a list, that list may be pre-selected.
- The dropdown options depend on your role: some users see all company lists, others only lists they are assigned to.
- Changing the selected list reloads the table with that list’s contacts.



## 👁️‍🗨️ Showing and hiding columns

Use **Show/Hide Columns** to control which optional columns are visible.

- Selection checkbox, row number, and action column stay visible.
- Optional contact fields can be shown/hidden in any combination.
- Preferences are saved in this browser for around 30 days.
- First-time/default behavior usually shows all optional columns.
- **Created At** can be toggled from this panel.



## 🗂️ What the table contains

**1️⃣ First column — selection**

- Row checkbox selects contact for bulk actions.
- Header checkbox selects/clears all rows on current page.

**2️⃣ Second column — row number**

- Simple row index.
- Can also be used in sorting flows where available.

**⚡ Third column — Action**

Per-row quick actions:

- **View** (eye): opens that contact’s conversation/activity view.
- **Delete** (trash): deletes that contact after confirmation.

Delete is permanent.

**🧑‍💼 Contact detail columns**

After Action, table shows your organization’s configured contact fields (for example name, phone, email, address, custom fields).

- Labels are displayed in readable form.
- Some internal system-only fields are hidden from table for clarity.

**🕑 Created At**

Shows timestamp in your account timezone. In this implementation, it reflects the contact’s last update time so you can judge record freshness.



## 🔍 Searching

Use the search box (placeholder: **Search Contact...**) to filter loaded contacts for the selected PowerList.

Search scope = current list context, not all lists globally.



## ↔️ Sorting

Default sort usually places newer/higher-index records first.

Where header sorting is available, click column headers to change sort order.



## 🗂️ Bulk Actions

Selecting at least one row shows a bulk-action bar.

<img
  src="/images/manage/contact/bulk-actions.png"
  alt="Bulk actions bar in Contacts"
  className="rounded-lg border shadow-sm"
/>

It includes three main actions:

**🔄 Change PowerList**

1. Select destination PowerList.
2. Click **Apply**.
3. Confirm.

Selected contacts are moved to the chosen list.

**🏷️ Update Tags**

1. Select one or more tags.
2. Click **Apply**.
3. Confirm.

Important behavior: this replaces existing tags on selected contacts with the chosen set.

**🗑️ Delete**

Click **Delete** and confirm.

All selected contacts are permanently removed.

Counts near action buttons show how many contacts are selected.
