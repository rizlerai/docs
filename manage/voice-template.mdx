# 🎤 Voice Templates

Create reusable voice templates for AI agents, automated voice calls, and professional audio scripts with customizable audio recordings.

## 📋 Template Overview
<img
  src="/images/manage/voice-template/overview.png"
  alt="SMS Templates Interface"
  className="rounded-lg border"
/>
View all your voice templates in one organized interface with audio previews, text-to-speech content, and easy management tools.

### 📊 Template Fields

| Field | Description | Example |
|-------|-------------|---------|
| **#** | Template ID | `1`, `2`, `3`, `4` |
| **Title** | Template name | `dev`, `textvoice`, `b35` |
| **Audio** | Voice recording player | Audio player with play/pause controls |
| **Text-To-Speech** | Script content | `hello this is a test call.` |
| **Status** | Template status | `active` |
| **Updated At** | Last modification | `04-14-2025 06:48 PM` |
| **Action** | Available operations | `Edit` `Delete` |

## 🔧 Voice Script Examples

Create professional scripts for different use cases:

### 📞 Call Opening
```
Hello {{customer_name}}, this is {{agent_name}} from {{company_name}}. 
I hope you're having a great day. I'm calling because you expressed 
interest in our {{product_name}}. Do you have a few minutes to chat?
```

### 🏢 Business Introduction
```
Hi there! Thank you for your interest in {{company_name}}. 
We specialize in {{service_type}} and have been helping businesses 
like yours for over {{years_experience}} years.
```

### ⏰ Appointment Reminder
```
This is a friendly reminder that you have an appointment with 
{{company_name}} tomorrow, {{appointment_date}} at {{appointment_time}}. 
Please call {{business_phone}} if you need to reschedule.
```

## 🔧 Dynamic Variables

Add personalization to your voice templates:

### 👤 Customer Info
- `{{customer_name}}` - Customer's full name
- `{{first_name}}` - Customer's first name
- `{{company}}` - Customer's company
- `{{phone_number}}` - Customer's phone

### 🏢 Business Info
- `{{company_name}}` - Your business name
- `{{agent_name}}` - AI agent name
- `{{business_phone}}` - Your phone number
- `{{website}}` - Your website

### 📅 Date & Time
- `{{current_date}}` - Today's date
- `{{current_time}}` - Current time
- `{{appointment_date}}` - Scheduled date
- `{{appointment_time}}` - Scheduled time

## ⚡ Audio Controls

### 🎛️ Audio Player Features
- **Play/Pause**: Control audio playback
- **Time Display**: Shows current position and total duration
- **Progress Bar**: Visual playback progress
- **Volume Control**: Adjust audio volume
- **Download**: Save audio files locally

### 🎚️ Audio Settings
- **Playback Speed**: Adjust from 0.5x to 2.0x speed
- **Audio Quality**: Choose quality settings
- **Format Options**: Select output audio format
- **Loop Options**: Enable/disable audio looping

## ⚡ Template Actions

### ✏️ Edit Template
Click the **Edit** button to modify:
