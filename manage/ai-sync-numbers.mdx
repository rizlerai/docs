This guide explains the **Phone Numbers** area in simple language, including list actions, filters, per-number actions, purchasing, editing, and bulk operations.

<img
  src="/images/manage/ai-sync-number/overview.png"
  alt="Phone Numbers main list"
  className="rounded-lg border shadow-sm"
/>

The **Phone Numbers** page is where you see every number tied to your workspace, buy new ones, and control how each number is used for AI calls, dialer workflows, and messaging compliance.

You can also:

- import external numbers,
- link 10DLC campaigns,
- configure endpoints for inbound messaging,
- perform bulk updates across rows.

Breadcrumb: `Users -> Phone Numbers`


### 🏁 Actions at the Top

- **Purchase:** Opens **Purchase Number** flow to search and buy one or many numbers.
- **Import Twilio Numbers:** (If enabled) Lets you add your own Twilio numbers.
- **Local Preference API:** Opens guidance for checking local presence (endpoint details shown).


### 🔍 Filters, Search, and Export

Filters can be combined. Editing any filter refreshes the table.

| Control      | What it does                                                                      |
| ------------ | --------------------------------------------------------------------------------- |
| **Filter By** | All, Ai, Dialer, Both                                                           |
| **Provider** | All, first carrier, Twilio, or V2                                                |
| **Type**     | All, Local, Toll Free, Mobile                                                    |
| **Status**   | All, Unverified, Verified                                                        |

Other tools:

- **Download:** Export current view to spreadsheet.
- **Search:** Box for searching by title/number.
- **Paging & Sort:** Set rows per page, change pages, sort by Purchased At (usually newest first).

### ☝️ Single-Number Actions (`...` menu)

Available actions (some may depend on provider/state):

- **Spam Protect:** Add number to spam protection.
- **Edit:** Update title/status/10DLC/use-case.
- **Activate for AI:** Connects number for AI calling/routing (may disappear after activation).
- **Delete:** Remove number after confirmation (provider cleanup applied if supported).
- **Endpoints:** Show inbound endpoint details for integrations.
- **10DLC:** Assign a messaging campaign (if not yet assigned).


### ⚡ Quick: Activate for AI

**One number:**

1. Go to **Users -> Phone Numbers**.
2. Open `...` menu on a row.
3. Click **Activate for AI**.
4. Confirm and check for messages.

**Multiple numbers:**

1. Select several rows.
2. Open **Bulk Action**.
3. Choose **Activate For AI**.
4. Confirm.

*After activation*: Number is enabled for AI calls/logic (system checks for required credentials; error if missing).

### 🛒 How to Purchase a Number

<img
  src="/images/manage/ai-sync-number/purchase.png"
  alt="Purchase search results table"
  className="rounded-lg border shadow-sm"
/>


Click **Purchase** to open the buy form.

<img
  src="/images/manage/ai-sync-number/new-purchase.png"
  alt="Purchase Number form"
  className="rounded-lg border shadow-sm"
/>

Steps:

1. **Provider:** Choose first carrier, Twilio, or V2.
2. **Countries:** Pick USA/Canada.
3. **Type:** Local, Toll Free, Mobile.
4. **Search By:** AreaCode, Region, City.
5. Enter value, click **Search**.

Table results:

| Column           | Meaning                                    |
| ---------------- | ------------------------------------------ |
| **#**            | Result order                               |
| **Friendly Name**| Label from provider                        |
| **Phone #**      | The candidate number                       |
| **Locality**     | Geo details (optional)                     |
| **Capabilities** | Voice/SMS/MMS supported?                   |
| **Checkbox**     | Select for bulk buy                        |
| **Action**       | **Purchase** (single row)                  |

Options:

- **Purchase:** Buys that number.
- **Buy Multiple:** Buys all selected.

(If required fields are missing, the form will prompt you. Billing/provider setup may also be needed.)


### ✏️ Edit a Number

Open `...` → **Edit** on a row to update:

| Field           | Meaning                                  |
| --------------- | ---------------------------------------- |
| **Title**       | Internal label                           |
| **Status**      | Verified / Unverified                    |
| **10DLC Enable**| Yes/No for 10DLC features                |
| **Use For**     | Both, Ai, or Dialer                      |

Click **Submit** to save.

### 📦 Bulk Actions

Select rows with checkboxes to see bulk action bar.

<img
  src="/images/manage/ai-sync-number/bulk-action.png"
  alt="Bulk actions on phone numbers list"
  className="rounded-lg border shadow-sm"
/>

Bulk actions:

- **Verified / Unverified:** Change status.
- **Spam Protect:** Attempt bulk spam-protection.
- **Activate For AI:** Activate AI where possible.
- **Delete:** Remove selected numbers after prompt.

Partial success is possible (system will report failures if some rows can't be processed).



### 📝 10DLC & Routing Notes

- **10DLC** mainly affects compliant US texting.
- Voice-only can work without 10DLC.
- SMS flows may route through other numbers if 10DLC isn't linked.

Assign 10DLC campaigns using the row action in the table.
