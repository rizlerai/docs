This guide explains the **Phone Numbers** area in simple language, including list actions, filters, per-number actions, purchasing, editing, and bulk operations.

<img
  src="/images/manage/ai-sync-number/overview.png"
  alt="Phone Numbers main list"
  className="rounded-lg border shadow-sm"
/>

The **Phone Numbers** page is where you see every number tied to your workspace, buy new ones, and control how each number is used for AI calls, dialer workflows, and messaging compliance.

You can also:

- import external numbers,
- link 10DLC campaigns,
- configure endpoints for inbound messaging,
- perform bulk updates across rows.

Breadcrumb: `Users -> Phone Numbers`


### 🏁 Actions at the Top

- **Purchase:** Opens **Purchase Number** flow to search and buy one or many numbers.
- **Import Twilio Numbers:** (If enabled) Lets you add your own Twilio numbers.
- **Local Preference API:** Opens guidance for checking local presence (endpoint details shown).


### 🔍 Filters, Search, and Export

Filters can be combined. Editing any filter refreshes the table.

| Control      | What it does                                                                      |
| ------------ | --------------------------------------------------------------------------------- |
| **Filter By** | All, Ai, Dialer, Both                                                           |
| **Provider** | All, first carrier, Twilio, or V2                                                |
| **Type**     | All, Local, Toll Free, Mobile                                                    |
| **Status**   | All, Unverified, Verified                                                        |

Other tools:

- **Download:** Export current view to spreadsheet.
- **Search:** Box for searching by title/number.
- **Paging & Sort:** Set rows per page, change pages, sort by Purchased At (usually newest first).

### ☝️ Single-Number Actions (`...` menu)

When you click the `...` menu on a number row, you'll see several possible actions, depending on the provider and state of the number:

- **Spam Protect:** Enable spam protection for the number.
- **Edit:** Change the title, status, 10DLC registration, or use-case for the number.
- **Activate for AI:** Enable AI features for calling and routing with this number (option might disappear after activation).
- **Delete:** Permanently remove the number, with provider-side cleanup if supported (confirmation required).
- **Endpoints:** View inbound API endpoint and integration details.
- **10DLC:** Assign or update a messaging campaign if not already set.

<img
  src="/images/manage/ai-sync-number/assign-chatbot.png"
  alt="Assign Chatbot to Number – agent selection popup"
  className="rounded-lg border shadow-sm"
/>

- **Assign Chatbot:**  
  As shown above, this lets you assign an AI (chatbot) to the number from a list of available agents. Once assigned, this AI agent will automatically respond to chat messages sent to the number and continue conversations with contacts. Use this option to connect a specific chatbot to handle communications for that number.

### ⚡ How to Quickly Activate for AI

**To activate AI for a single number:**

1. Navigate to **Users → Phone Numbers**.
2. Click the `...` menu for the relevant number.
3. Select **Activate for AI**.
4. Confirm the action and then look for a success message.

**To activate AI for multiple numbers:**

1. Select the numbers you want using the checkboxes.
2. Click on the **Bulk Action** bar.
3. Choose **Activate For AI**.
4. Confirm your choice.

After activation, numbers are enabled for AI calls and logic. If required credentials or setup are missing, the system will display an error message.

### 🛒 Purchasing a Phone Number

<img
  src="/images/manage/ai-sync-number/purchase.png"
  alt="Purchase search results table"
  className="rounded-lg border shadow-sm"
/>

Click the **Purchase** button to start the buying process.

<img
  src="/images/manage/ai-sync-number/new-purchase.png"
  alt="Purchase Number form"
  className="rounded-lg border shadow-sm"
/>

Follow these steps:

1. **Provider:** Choose the carrier (first carrier, Twilio, or V2).
2. **Countries:** Select the country (USA/Canada).
3. **Type:** Pick the number type—Local, Toll Free, or Mobile.
4. **Search By:** Filter by AreaCode, Region, or City.
5. Enter a value and click **Search**.

Your search results will display a table like this:

| Column           | Description                                 |
| ---------------- | ------------------------------------------- |
| **#**            | Row/order number                            |
| **Friendly Name**| Provider-supplied label                     |
| **Phone #**      | The available number                        |
| **Locality**     | Geographic information (if shown)           |
| **Capabilities** | Whether the number supports Voice/SMS/MMS   |
| **Checkbox**     | Select for multi-buy                        |
| **Action**       | **Purchase** (to buy instantly)             |

You can:

- **Purchase** a single number by clicking its action button.
- **Buy Multiple** numbers at once by selecting checkboxes and confirming.

If any required details are missing, you'll see a prompt before purchase completes. Billing and provider setup may also be required before buying.


### ✏️ Edit a Number

Open `...` → **Edit** on a row to update:

| Field           | Meaning                                  |
| --------------- | ---------------------------------------- |
| **Title**       | Internal label                           |
| **Status**      | Verified / Unverified                    |
| **10DLC Enable**| Yes/No for 10DLC features                |
| **Use For**     | Both, Ai, or Dialer                      |

Click **Submit** to save.

### 📦 Bulk Actions

Select rows with checkboxes to see bulk action bar.

<img
  src="/images/manage/ai-sync-number/bulk-action.png"
  alt="Bulk actions on phone numbers list"
  className="rounded-lg border shadow-sm"
/>

Bulk actions:

- **Verified / Unverified:** Change status.
- **Spam Protect:** Attempt bulk spam-protection.
- **Activate For AI:** Activate AI where possible.
- **Delete:** Remove selected numbers after prompt.

Partial success is possible (system will report failures if some rows can't be processed).



### 📝 10DLC & Routing Notes

- **10DLC** mainly affects compliant US texting.
- Voice-only can work without 10DLC.
- SMS flows may route through other numbers if 10DLC isn't linked.

Assign 10DLC campaigns using the row action in the table.
