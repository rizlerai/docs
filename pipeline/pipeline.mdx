The Pipeline module helps you organize your business processes by creating pipelines with multiple stages. Each pipeline represents a workflow, and each stage represents a step in that workflow. Stages appear as columns on your board, allowing you to visualize and manage your processes. This system makes it easier to track progress through different stages of your business operations.

## ‚ú® Main Features

### üóÉÔ∏è Pipeline Board View

When you open the Pipeline page and select a pipeline, you see a visual board that displays the stages of your selected pipeline as columns.

**What you see:**

- Multiple columns displayed horizontally across your screen
- Each column represents one stage from your pipeline
- Each column shows a header with:
  - The stage name at the top
  - A number badge showing the count of contacts in that stage
  - A dollar amount showing the total opportunity value for that stage
- Contact cards displayed within each column representing contacts at that stage

**How it works:**

- When you select a pipeline from the dropdown, the board shows only the stages belonging to that pipeline
- Each column is a separate stage
- You can scroll up and down within each column
- Each column is scrollable independently
- The columns appear in the order you set when creating or editing the pipeline
- Contact cards can be dragged and dropped between columns to move contacts through stages

### üîΩ Pipeline Selection

At the top of the page, you will find a dropdown menu that lets you choose which pipeline you want to view.

**How it works:**

- Click on the dropdown menu
- Select the pipeline you want to see
- The board will automatically update to show only the stages belonging to that pipeline
- Each stage appears as a column on the board
- The most recently created pipeline is automatically selected when you first open the page
- When you switch pipelines, the board changes to show the stages of the newly selected pipeline

### üõ†Ô∏è Pipeline Management

You can create, edit, and delete pipelines through the Pipeline Management feature.

<img
  src="/images/pipeline/pipeline-management.png"
  alt="Pipeline Management Interface"
  className="rounded-lg border shadow-sm"
/>

**How to access Pipeline Management:**

1. Click the "Pipeline Management" button at the top of the page
2. A popup window opens showing a list of all your pipelines

**Viewing Pipelines:**

The list shows:

- **#**: Pipeline number
- **Pipeline Name**: Name of the pipeline
- **Description**: Pipeline description
- **Dispositions**: Associated dispositions (comma-separated)
- **Status**: Active/Inactive status badge
- **Created At**: Date when the pipeline was created
- **Actions**: Edit and Delete buttons

**Creating a New Pipeline:**

<img
  src="/images/pipeline/add-pipeline.png"
  alt="Add Pipeline Form"
  className="rounded-lg border shadow-sm"
/>

1. Click the "+ Add Pipeline" button in the Pipeline Management window
2. A new popup opens for creating a pipeline
3. Enter a name for your pipeline in the "Pipeline Name" field (required)
4. Add a description in the "Description" field (optional)
5. Select dispositions from the "Dispositions" dropdown (required)
6. Review the status (defaults to "Active")
7. Add stages to your pipeline:
   - Click the "Add Stage" button to add a new stage
   - Enter a name for each stage
   - You can add as many stages as you need
   - You can rearrange stages by dragging them up or down
   - You can delete stages by clicking the delete button (at least one stage must remain)
8. Click "Save Pipeline" to create it
9. The page will refresh and your new pipeline will appear in the dropdown menu

**Editing a Pipeline:**

1. Click the Edit button (pencil icon) next to the pipeline you want to modify
2. The pipeline details open in a popup
3. You can:
   - Change the pipeline name
   - Update the description
   - Modify dispositions
   - Change status
   - Add new stages
   - Edit existing stage names
   - Delete stages (except the last one)
   - Reorder stages by dragging them
4. Click "Save Pipeline" to save your changes
5. The page will refresh to show your updated pipeline

**Deleting a Pipeline:**

1. Click the Delete button (trash icon) next to the pipeline you want to remove
2. A confirmation message appears asking if you're sure
3. Click OK to confirm deletion
4. The pipeline is permanently removed
5. The page refreshes and the pipeline is no longer available

### üîé Filtering and Display

When you select a pipeline from the dropdown, the board automatically shows only the stages belonging to that pipeline.

**How it works:**

- Only stages that belong to the selected pipeline are displayed as columns
- Stages from other pipelines are not shown
- Each stage appears as a separate column on the board
- The stages appear in the order you set when creating or editing the pipeline
- This keeps your view clean and focused on the pipeline you're working with

### üîÑ Data Updates

The system automatically updates information in real-time as you make changes.

**Automatic Updates:**

- When you modify pipeline stages, the board updates after page refresh
- When you reorder stages, the changes are automatically saved
- When you edit stage names, the changes are saved when you click "Save Pipeline"
- When you create or delete pipelines, the dropdown menu updates automatically
- Contact counts and opportunity values update when contacts move between stages
- Stage statistics recalculate in real-time as cards are moved
