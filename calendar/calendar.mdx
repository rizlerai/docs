## My Calendars

The "My Calendars" section displays all your created calendars in an organized card-based layout. Each calendar card provides essential information at a glance and quick access to management options.

### Calendar Card Information

Each calendar card displays:

- **Calendar Icon**: A colored circular icon representing the calendar
- **Calendar Name**: The name you assigned to the calendar 
- **Type Tag**: A colored tag indicating the calendar type:
  - **Personal** : For individual appointments and personal scheduling
  - **Company**: For business meetings and corporate events
- **Description**: Brief details about the calendar's purpose (e.g., "Testing", "No description")
- **Status Tags**: Important indicators showing calendar functionality:
  - **Booking Enabled** : Users can book appointments on this calendar
  - **Google Connected**: Calendar is synced with Google Calendar
- **Action Menu**: Three-dot menu (•••) in the top-right corner for additional options

![calendar](/images/calendar/image3.png)

### Creating a New Calendar

When you click the "+ Create Calendar" button, a modal will open with the following options:

- **Calendar Name** (required): Enter a name for your calendar.
- **Type**: Select the calendar type (e.g., Personal, Company).
- **Description**: Add optional details about the calendar's purpose.
- **Color**: Choose a color to help visually distinguish this calendar.
- **Booking Settings**: Enable "Allow slot booking" to let users book time slots on this calendar.

Click **Create Calendar** to save your new calendar, or **Cancel** to close the modal without saving.

![calendar](/images/calendar/image8.png)

### Calendar Actions

Click the three-dot menu (•••) on any calendar card to access:

- **Manage Calendar**: Opens the comprehensive calendar management interface with five tabs (Overview, Settings, Availability, Booking, Integration)
- **View Events**: Shows all scheduled events and appointments for the selected calendar
- **View Bookings**: Displays all pending and confirmed bookings
- **Edit Calendar**: Quick access to modify basic calendar information
- **Delete Calendar**: Remove the calendar and all associated data (requires confirmation)

![calendar](/images/calendar/image4.png)

### Calendar View Integration

The right panel "Calendar View" works in conjunction with My Calendars:

- **Select Calendar**: Choose which calendar to display in the main view
- **Event Display**: View all events and bookings for the selected calendar
- **Empty State**: When no calendar is selected, shows helpful guidance to choose a calendar

![calendar](/images/calendar/image5.png)



## Managing Calendars

To manage an existing calendar, click the menu (•••) on any calendar and select "Manage Calendar". This opens a detailed management interface with five tabs:(Overview, Settings, Availability, Booking, Integration)

![calendar](/images/calendar/image10.png)

### Overview

The Overview tab provides a comprehensive view of your calendar:

- **Calendar Information**: Shows the calendar name, description, and status tags (Personal/Company, Booking Enabled, Google Connected)
- **Calendar Events**: Displays a full calendar grid showing all events and bookings
- **Navigation Controls**: Use arrows to navigate between months, "today" button to return to current month, and view toggles (month/week/day)
- **Quick Stats**: Shows total events, monthly count, pending bookings, and confirmed bookings
- **Quick Actions**: 
  - "Next Available Slot" button to find the next open time
  - "View All Bookings" button to see all scheduled appointments

![calendar](/images/calendar/image9.png)

### Settings

Configure your calendar's basic settings:

- **Calendar Name & Color**: Edit the calendar name and choose a display color
- **Timezone**: Set your calendar's timezone (e.g., Winnipeg / Canada / CST / CT)
- **Description**: Add or edit calendar description
- **Booking & Activity Controls**:
  - "Allow slot booking" - Enable appointment scheduling
  - "Calendar active" - Enable/disable the calendar
  - "Available 24/7" - Override working hours (optional)
- **Appointment Permissions**: Choose who can book appointments (internal only, public, or both)
- **Calendar URL**: View and copy your public booking link
- **Action Buttons**: Save changes or delete the calendar

![calendar](/images/calendar/image11.png)

### Availability

Set your working hours and availability:

- **Working Hours**: Configure availability for each day of the week
  - Check "Available" to enable each day
  - Set start and end times (e.g., 9:00 AM - 5:00 PM)
  - Option to set "Available 24 hours" for specific days
- **Current Schedule**: View a summary of your configured working hours
- **Default Schedule**: Monday-Friday 9:00-17:00, Sunday typically unavailable

![calendar](/images/calendar/image12.png)

### Booking

Configure appointment booking settings:

- **Slot Configuration**:
  - **Slot Duration**: Set appointment length (e.g., 30 minutes)
  - **Buffer Time**: Add time between appointments for preparation/travel
  - **Maximum Advance Booking**: Set how many days in advance people can book (e.g., 30 days)
  - **Minimum Advance Booking**: Set minimum notice required (e.g., 24 hours)
- **Booking Preview**: See how your configured slots will appear to users
- **Save Configuration**: Apply your booking settings

![calendar](/images/calendar/image13.png)

### Integration

Manage Google Calendar integration:

- **Connection Status**: Shows if connected to Google Calendar with account email
- **Enabled Features**:
  - Events automatically sync to Google Calendar
  - Google Meet links auto-generated
  - Calendar events visible in Google Calendar
- **Actions**: Disconnect from Google Calendar if needed

![calendar](/images/calendar/image14.png)
