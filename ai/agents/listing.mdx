## Folder Management System

**Sidebar Navigation**
- **All Agents** - View complete agent inventory across all folders
- **Custom Folders** - Organize agents by project, client, or purpose
- **+ Add** - Create new folders with custom names

## AI Number Management 

### Number Assignment Interface
The phone number management system provides comprehensive control over available and assigned phone numbers for AI agents, enabling seamless communication channel management.

![Phone Number Management Interface](/images/ai/AI-agent/listing/number-management.png)

### Available Numbers Section
**Left Panel - Available Numbers**
- **Number Display** - Shows available phone numbers (e.g., 205-203-2134, 205-204-2415)
- **Status Indicators** - "Green" status indicates numbers have spam protection
- **Action Buttons**:
  - **Outbound** - Assign number for making outgoing calls from AI agents
  - **Inbound** - Assign number for receiving incoming calls to AI agents

### Assigned Numbers Section  
    
**Right Panel - Assigned Numbers**
- **Number Tracking** - Displays currently assigned phone numbers
- **Type Classification** - Shows number purpose (Outbound/Inbound designation)
- **Status Monitoring** - Real-time status updates for assigned numbers
- **Management Actions**:
  - **Status** - "Green" indicators for spam protected numbers
  - **Unassign** - Red button to remove number assignments and return to available panel

### Assignment Workflow
1. **Select Available Number** - Choose from left panel available numbers
2. **Configure Type** - Click "Outbound" or "Inbound" based on intended use
3. **Automatic Assignment** - Number moves to assigned panel upon configuration
4. **Status Updates** - Real-time reflection of assignment changes
5. **Unassign Process** - Use "Unassign" button to return numbers to available panel
