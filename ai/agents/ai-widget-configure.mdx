## Widget Configuration Interface

### Access Widget Configuration
- **Agent Actions Menu** - Click the action button (â‹®) next to any agent
- **Select AI Widget Configure** - Choose from the dropdown menu
- **Configuration Panel** - Opens comprehensive widget customization interface

![AI Dashboard Interface](/images/ai/dashbaord-widget-configure.png)
### Embed Code Section
- **JavaScript Integration** - Ready-to-use embed code for website integration
- **Copy Code** - Click the copy button to get the embed script
- **Agent-Specific** - Each agent has unique embed code with agent ID
- **Async Loading** - Non-blocking script for optimal website performance

![AI Dashboard Interface](/images/ai/dashbaord-embed-code.png)
## Avatar Configuration

### Avatar Upload
- **Upload Avatar Image** - Custom avatar image for the widget
- **Choose File** - Select image file from local computer
- **File Support** - Common image formats (PNG, JPG, GIF)
- **Avatar Display** - Shows in widget conversation interface

![AI Dashboard Interface](/images/ai/dashbaord-embed-avatar.png)
## Appearance Customization

- **Background Color** - Widget background color 
- **Text Color** - Main text color for widget content 
- **Button Color** - Primary button background color 
- **Button Text Color** - Button text color 
- **Border Color** - Widget border and outline color 
- **Focus Outline Color** - Highlight color for focused elements 
- **Card Radius** - Border radius for widget container 
- **Button Radius** - Border radius for buttons 
- **Popup Position** - Widget placement on webpage 

![AI Dashboard Interface](/images/ai/dashbaord-apperance.png)
## Text Contents Configuration

### Button Text Customization
- **Start Call Button** - Text displayed on call initiation button (default: "Start a call")
- **End Call Button** - Text shown when call ends (default: "Call ended")
- **Call to Action** - Header text to engage users (default: "Need help?")
- **Speaking Status** - Text displayed during active calls (default: "Agent is speaking")

![AI Dashboard Interface](/images/ai/dashbaord-embed-content.png)
