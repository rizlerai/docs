Simple and efficient AI agent testing through direct call initiation.

## Test Call Interface

### Access Test Call
- **Agent Actions Menu** - Click the action button (⋮) next to any agent
- **Select Test Call** - Choose "Test Call" from the dropdown menu
- **Test Call Dialog** - Opens dedicated test call interface

![AI Dashboard Interface](/images/ai/dashbaord-test-call.png)

## How to Test Call

### Step-by-Step Process
1. **Navigate to Agent List** - Go to the All Agents view
2. **Select Agent** - Find the agent you want to test
3. **Open Actions Menu** - Click the three dots (⋮) in the Action column
4. **Choose Test Call** - Select "Test Call" from the dropdown
5. **Enter Phone Number** - Input the destination phone number
6. **Start Test** - Click the "Call" button to initiate the test call
