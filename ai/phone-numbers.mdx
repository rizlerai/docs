Manage your AI phone numbers with comprehensive controls for SIP trunks, Rizz Numbers, and Twilio integrations.

## AI Numbers Overview

The AI Numbers interface provides complete management of your phone number infrastructure:

- **Multiple Number Types** - Manage SIP trunks, Rizz Numbers, and purchase Twilio numbers
- **Phone Number Table** - View all numbers with detailed information and status
- **Action Controls** - Edit, protect, manage tokens, and delete phone numbers
- **Search & Filter** - Find specific numbers and control display options

## Number Management Tabs

### Available Number Types
- **SIP trunks** - Manage SIP trunk connections and configurations
- **Rizz Numbers** - View and manage Rizz platform phone numbers  
- **Purchase Twilio No** - Acquire new Twilio phone numbers for your system

## Phone Numbers Table

### Table Columns
- **#** - Sequential number for each phone number entry
- **Name** - Custom identifier/name for the phone number (e.g., "devSIPDUP", "gmeet")
- **Phone Number** - Full phone number including country code (e.g., "+13156663670")
- **Spam Status** - Current spam protection status ("Clean" indicates good standing)
- **Area Code** - Geographic area code for the phone number (e.g., "315")
- **Renewal At** - When the phone number subscription renews
- **Updated At** - Last modification timestamp (e.g., "05-19-2025 11:29 PM")
- **Action** - Available management actions for each number

### Action Buttons
- **Spam Protect** - Enable or manage spam protection settings
- **Edit** - Modify phone number settings and configuration
- **Token** - Manage authentication tokens for the phone number
- **Delete** - Remove the phone number from your system

## Table Features

### Display Controls
- **Show Entries** - Select 10, 25, 50, or 100 entries per page
- **Search Functionality** - Search across all phone number data
- **Entry Counter** - Shows "1 to 2 of 2 entries" for current view status

### Navigation
- **Pagination Controls** - Previous/Next buttons and numbered pages
- **Current Page Indicator** - Highlighted page number (currently showing page 1)

## Phone Number Status
