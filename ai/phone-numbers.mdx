Manage your AI phone numbers with comprehensive controls for SIP trunks, Rizz Numbers, and Twilio integrations.

## AI Numbers Interface

The AI Numbers section provides centralized management of your phone number infrastructure:

- **Multi-Platform Support** - Manage SIP trunks, Rizz Numbers, and Twilio phone numbers
- **Real-Time Status** - Live spam status monitoring and system health indicators  
- **Action Management** - Complete control over phone number configuration and security
- **Search & Filter** - Advanced search capabilities for phone number management


## Management Tabs

### Available Platforms
- **SIP trunks** - Currently active tab for SIP trunk phone number management
- **Rizz Numbers** - Native platform phone numbers and configurations  
- **Purchase Twilio No** - Acquire and configure new Twilio phone numbers

## SIP Trunk Configuration

Click **Edit** to open the SIP trunking configuration dialog:

### Required Fields
- **Phone Number** - Number to configure
- **Termination URI** - SIP routing endpoint
- **SIP Trunk User Name** - Authentication username
- **SIP Trunk Password** - Authentication password
- **Inbound Agent** - AI agent for incoming calls
- **Outbound Agent** - AI agent for outgoing calls

### Optional
- **Nickname** - Custom identifier for the trunk

### Actions
- **Cancel** - Close without saving
- **Save** - Apply configuration

![SIP Trunk Configuration Interface](/images/ai/AI-numbers-SIP-trunk.png)

## Rizz Number Configuration

Click **Rizz Numbers** tab to access native platform number configuration:

### Required Fields
- **Setup Number** - Select from available Rizz numbers dropdown
- **Inbound Agent** - AI agent for incoming calls
- **Outbound Agent** - AI agent for outgoing calls

### Optional
- **Nickname** - Custom identifier for the number

### Actions
- **Cancel** - Close without saving
- **Save** - Apply configuration


![Rizz Number Configuration Interface](/images/ai/AI-numbers-rizz-number.png)

## Twilio Number Purchase

Click **Purchase Twilio No** to buy new numbers:

### Fields
- **Nickname** - Optional identifier
- **Inbound Agent** - AI agent for incoming calls
- **Outbound Agent** - AI agent for outgoing calls
- **Countries** - Select country (USA)
- **Search By** - AreaCode filter
- **AreaCode/Region/City** - Search criteria

### Actions
- **Search** - Find available numbers
- **Cancel** - Close without purchasing
- **Purchase Number** - Buy selected number

![Twilio Number Purchase Interface](/images/ai/AI-numbers-purchase-number.png)

## Phone Numbers Table

### Table Structure
The table displays all configured phone numbers with the following columns:

- **#** - Sequential number identifier for each entry
- **Name** - Custom identifier/label for the phone number
- **Phone Number** - Complete phone number with country code (+1)
- **Spam Status** - Current spam protection and reputation status
- **Area Code** - Geographic area code designation
- **Renewal At** - Phone number subscription renewal date
- **Updated At** - Last modification timestamp with date and time
- **Action** - Available management actions for each phone number

![Phone Numbers Table Interface](/images/ai/AI-numbers-table.png)


## Action Management

### Available Actions
- **Spam Protect**  - Configure spam protection settings and monitoring
- **Edit** - Modify phone number configuration and settings
- **Token**  - Manage API tokens and authentication credentials
- **Delete** - Remove phone number from system (permanent action)

![Phone Number Actions Interface](/images/ai/AI-numbers-actions.png)
