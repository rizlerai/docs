#### 🎛️  General Tab

This is where you set your AI agent’s **name**, **on or off state**, **folder**, **location**, **time zone**, **how the first message works**, and **full instructions (the prompt)**. **General** holds the core setup.



## 🗂️ Above the tabs

| Element | What it does |
|--------|----------------|
| **Back to Agents** | Returns you to the list of agents. |
| **Agent title (page heading)** | Shows the name currently used for this agent. |
| **Delete** | After confirmation, removes this agent. |
| **Talk With Agent** | Starts a test call so you can hear how the agent responds. **Hangup** ends the test call. |

**🏷️ Display name (agent name)**

You see the name and an **edit** control.

- **Edit** opens a field below; **Update** sends the new name.
- On success, the name at the top of the page and in General both update.
- The name is also updated on the calling side so calls use the same display name.

**🔛 Status**

A switch for **on** or **off**.

- **On** — the agent is treated as available for use.  
- **Off** — the agent is not active for handling work.

**📁 Folder**

Help text explains that you can group agents into folders for organization.

Choosing a folder **saves immediately** when you change the selection.

**📍 GHL Location**

Help text explains that this is the **GoHighLevel location** where call data, contacts, and conversation records are tied.

Changing the location **saves** and can refresh related choices elsewhere (for example calendars or field lists). On first load or when you change location, **automatic field matching** may run if no mappings existed before.

**🕒 Timezone**

Help text explains that this is the time zone used for this agent for scheduling and time-related behavior.

Changing the time zone **saves immediately**.



## 2️⃣ Welcome Message

Help text explains that you control **how the conversation starts** — whether the caller speaks first or the AI speaks first.

<img
  src="/images/ai/AI-agent/edit/general/welcome-message.png"
  alt="General prompt area with sectioned and single modes"
  className="rounded-lg border shadow-sm"
/>

Three choices:

1. **User initiates**  
   The AI stays quiet until the caller speaks.

2. **AI initiates — dynamic opening**  
   The AI starts the call; the first line comes from the **system default** (you do not type a custom line).

3. **AI initiates — your own opening**  
   The AI starts and uses **exactly the text** you type in the box below.

The large text box **only appears** when the third option is selected.

These welcome settings are applied when you press **Save** on the prompt (see below), together with the rest of the General prompt.



## 3️⃣ General Prompt

Help text explains that this is where you define **how the agent behaves**, **what it should do**, and **what limits it should follow**. It is the foundation for conversations.

<img
  src="/images/ai/AI-agent/edit/general/prompt-select.png"
  alt="General prompt area with sectioned and single modes"
  className="rounded-lg border shadow-sm"
/>

**📝 Default Prompt**

Opens a small window: choose an **industry**, then a **starter prompt**. **Use This Prompt** fills the related fields; you can edit afterward..

**🏷️ Prompt name**

A **name is required** before saving.

When saved, the system adds a **date and time** to the name so each saved copy is distinct.

**📜 Version list (drop-down)**

Shows saved prompt versions for **this agent**.

Choosing one **reloads** those sections into the form.

The trash control **deletes** the selected version, after confirmation.

**🗃️ Sectioned vs Single**

- **Sectioned Prompt** — one box per topic (easier editing).  
- **Single Prompt** — everything in one large box (for users who prefer one canvas).

When you switch modes, the content is **split** or **merged** so nothing is lost.

**📋 Copy, Paste, Clear**

- **COPY PROMPT** — copies the active content (section format when in sectioned mode).  
- **PASTE PROMPT** — pastes that format back into the fields.  
- **CLEAR PROMPT** — clears the fields, after confirmation.

A short line under the radios explains the current mode in plain language.



## 4️⃣ Placeholder tags (dynamic variables)

Below the prompt area, a line shows **tags in curly braces**. These come from your contact fields, plus **current date and time** and **join meeting pin** where applicable.

During a live call, these tags are **filled with real values** for that contact or call so the agent can speak accurately.


## 5️⃣ Sectioned boxes (when Sectioned mode is on)

Each section has a help icon with a short hint.

| Section | Purpose in plain words |
|---------|-------------------------|
  | **Project instructions / Request** | What you want the agent to accomplish — the main job. |
| **Greetings** | How every conversation should start. |
| **Call Flow** | The natural order of the call. |
| **Character** | Tone, personality, style. |
| **Transfer Call** | How to hand off to a human. |
| **Critical Instructions / Guardrails** | What must never happen; boundaries. |
| **Custom Field References** | Notes on extra fields you rely on. |
| **What Your Company Does** | What you offer and how you help. |
| **Script** | Step-by-step talking points. |
| **Objection Handling** | How to respond to pushback. |
| **Booking flow** | How appointments should be booked (including with GHL). |
| **FAQ / Knowledge Base** | Common questions and facts. |

Each section has **Expand** to enlarge the box for long text.

In **Single** mode, one **All Prompts** box is shown instead.



## 6️⃣ Apply this prompt to other agents (when shown)

This block appears when the system offers **other agents** you can target.

<img
  src="/images/ai/AI-agent/edit/general/to-other-agent.png"
  alt="Save and version controls"
  className="rounded-lg border shadow-sm"
/>


- You may select **more than one** agent (hold Ctrl on Windows or Command on Mac while clicking).
- When you press **Save**, the same prompt and welcome behavior can be applied to those agents so you do not repeat the work for each one.

Some agent types may be updated through a different path behind the scenes; the outcome for you is still one save that updates multiple agents when selected.


## 7️⃣ Save and New Version

<img
  src="/images/ai/AI-agent/edit/general/save.png"
  alt="Save and version controls"
  className="rounded-lg border shadow-sm"
/>

**💾 Save**

- **Prompt name** is required.  
- If **no older version** is selected in the list, the save is treated as a **new copy** under that name.  
- If **an older version is selected**, you are asked whether to **replace that version** or **create another copy**.  
  - **Override** — that version is updated with your current text.  
  - **New version** — the old row stays; a new row is added.

When save succeeds:

<img
  src="/images/ai/AI-agent/edit/general/save-functions.png"
  alt="Override and related save options"
  className="rounded-lg border shadow-sm"
/>

- All sections are **combined into one full instruction** for the AI, with a **current date and time** line at the end.  
- That full instruction is what the **live agent** uses.  
- A **history row** is also stored with a timestamp in the name.

**🆕 New Version**

If nothing is selected in the version list, this path creates a new version directly.

If something is selected, you may be asked for a name, then a new version is stored.
