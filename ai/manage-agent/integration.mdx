# 🔗 Integrating with GoHighLevel (GHL)

Seamlessly connect your AI agent to GoHighLevel to automate CRM updates with call recordings, transcriptions, analytics, and tags—no manual data entry required.

![Integration Overview](/images/ai/AI-agent/edit/integration/overview.png)

## 📤 Recording & Transcription Sync

Configure which call details are sent to GHL:

- **Recording Upload**  
  Automatically send call recordings to GHL, or toggle off if not needed.
- **Transcription Upload**  
  Choose whether AI-generated call transcripts should be added to GHL.
- **User Sentiment Sharing**  
  Send AI-scored sentiment (positive, neutral, negative) for each call.
- **Call Summary Sync**  
  Populate GHL custom fields with concise summaries of each call.

*First, create custom fields in GoHighLevel for details like "Recording URL," "AI Transcription," "Sentiment," and "Summary." In the integration dashboard, map each AI data point to your chosen GHL field. Once mapped, data flows automatically.*

![Recording and Transcription Integration](/images/ai/AI-agent/edit/integration/image1.png)

## ⏱️ Call Analytics & Data Points

Send key analytics to GHL:

- **Call Duration**  
  Automatically log call duration for tracking or billing.
- **Call Direction (Inbound/Outbound)**  
  Record whether each call was incoming or outgoing.

*For every metric, add matching custom fields in GHL (e.g., "Call Duration"), then connect your agent’s analytics to these fields within the integration settings.*

![Call Data Integration](/images/ai/AI-agent/edit/integration/image2.png)

## 🏷️ GHL Call Outcome Tags

Let your AI agent tag contacts/leads in GHL based on call results:

- **Answered**
- **No Answer**  
- **Busy**
- **Inactivity**
- **Appointment Set**
- **Call Transfer**
- **Voicemail**
- **Call Direction (Inbound/Outbound)**
- **User Hung Up**
- **Failed Outbound Call**

For each outcome, the relevant tag is automatically applied—helping you filter, track, and automate next actions in GHL.

> **Tip:** Tags appear immediately in your GHL CRM for rapid sorting, follow-up automation, and reporting—no extra setup needed.
