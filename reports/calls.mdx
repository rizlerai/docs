# 📞 Call Report (Reports → Call)

The **Call** option under **Reports** is where you review **phone calls** for your organization: who was involved, numbers, outcomes, length, and when each call happened. Everything below is about that **Call** report only.

<img
  src="/images/reports/call/overview.png"
  alt="Call report table and filters"
  className="rounded-lg border shadow-sm"
/>


**👀 What you see here**

**Phone calls** show in a table—**one row per call**. The table updates when you change filters and apply them, or when you change the date range (where that refreshes on its own).

At the top you pick **which time range** and **which calls** matter. Each row below is **one call** that matches those choices.


## 🛠️ Filters

<img
  src="/images/reports/call/filters.png"
  alt="Call report filter controls"
  className="rounded-lg border shadow-sm"
/>

 **🗓️ Date range**

Controls how **Start Date** and **End Date** are filled:

- **Custom dates** — You enter **Start Date** and **End Date** yourself; those fields are editable.
- **Presets** (*Today*, *Yesterday*, *This week*, *Last 7 days*, *This month*, *Last month*, and similar) — The dates are set for you and stay fixed for that preset.

If you are **not** on custom dates, changing the date range refreshes the **calls** shown. On **custom dates**, set start and end, then use **Filter** (or another control that refreshes).

Call times use **your account time zone**, so “today” matches your local calendar day.

**📅 Start date and end date**

Work with **Date range**. They set the **from–to** window for **calls** in the table. Editable only when **Custom dates** is selected.

**✅ Call status**

Limits **calls** to one outcome, or shows **all** outcomes:

- **All** — Every outcome.
- **Answered**, **Completed**, **Busy**, **No-answer**, **Failed**, **Missed Call**, **Cancel** — Only **calls** with that outcome.

Pick a value and apply it (for example **Filter**, or change the date range where it auto-refreshes).

**⏱️ Minimum duration**

Keeps **calls** whose **length** is **at least** what you select (from zero up to two hours). Shorter **calls** are hidden—useful when you only care about **calls** that ran long enough.

**⏳ Maximum duration**

Optional upper cap on length:

- **No limit** — No maximum.
- Any other value — **Calls** **longer** than that are excluded.

Together with minimum duration, you can focus on **calls** in a certain length range.

**🔄 Call direction**

- **All** — Incoming and outgoing **calls**.
- **Inbound** — **Calls** you received.
- **Outbound** — **Calls** you placed.

**👤 Agents**

Shows **calls** for **one** person, or—where the menu includes it—**everyone** (**All**). The names are people who can place or receive **calls** in your setup.

If the team is not set up, you may see a single placeholder instead of names.

**🔎 Filter**

Applies your choices and reloads the **call** table. Use it after changing something that does not refresh by itself (for example **custom** start and end dates).

**💾 Download (export)**

Saves a file using the **same** choices as the Call Report (dates, status, duration limits, direction, agent). The file has contact name, numbers, transcription, status, direction, how long the **call** lasted, and when it happened—so you can open it in a typical office spreadsheet program.

## 📋 What each column means (calls)

<img
  src="/images/reports/call/history-id.png"
  alt="Call report row example"
  className="rounded-lg border shadow-sm"
/>

**#️⃣ Sr. No.**

A reference number for the row. Newer calls appear higher up.

**🧑 First name / Last name**

The **contact** linked to that **call**, when known. A dash means the name was not available.

**📞 From number**

Where the **call** came from (caller side).

**☎️ To number**

Where the **call** went (destination).

**📝 Transcription**

Text from the **call**, when available. Long text may wrap so you can read it in the same row.

**🏷️ Status**

**How the call ended**, plus:

- A small **incoming** or **outgoing** marker for quick direction.
- For completed calls with a **recording**, a **player** so you can listen without leaving the Call Report.

<img
  src="/images/reports/call/audio.png"
  alt="Call recording player on a call row"
  className="rounded-lg border shadow-sm"
/>

**🗓️ Dated** 

When that **call** was logged, in **your** time zone.
