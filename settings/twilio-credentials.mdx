# Twilio Credentials

Configure your Twilio account credentials and communication service settings for SMS, voice calls, and messaging features.

## Account Configuration

### Primary Credentials
- **Account SID** - Your Twilio Account Security Identifier
- **Auth Token** - Authentication token for API access
- **API Key** - Optional API key for enhanced security
- **API Secret** - Corresponding API secret for key-based authentication
- **Subaccount SID** - Configure subaccounts for organizational separation

### Connection Settings
- **Region** - Select Twilio region for optimal performance
- **Edge Locations** - Configure edge locations for global connectivity
- **Webhook URL** - Set up webhook endpoints for status updates
- **Status Callback** - Configure status callback URLs for message delivery
- **Fallback URL** - Backup webhook URL for high availability

## Voice Services Configuration

### Voice Settings
- **Voice URL** - Primary webhook URL for incoming voice calls
- **Voice Method** - HTTP method (GET/POST) for voice webhooks
- **Voice Fallback URL** - Backup URL for voice call handling
- **Voice Status Callback** - URL for voice call status updates
- **Voice Recording** - Enable automatic call recording

### Call Features
- **Caller ID** - Configure outbound caller ID numbers
- **Call Forwarding** - Set up call forwarding rules and destinations
- **Conference Calling** - Enable conference call capabilities
- **Call Screening** - Configure call screening and filtering
- **International Calling** - Enable international calling permissions

## SMS and Messaging

### SMS Configuration
- **SMS URL** - Webhook URL for incoming SMS messages
- **SMS Method** - HTTP method for SMS webhooks
- **SMS Fallback URL** - Backup URL for SMS handling
- **SMS Status Callback** - URL for SMS delivery status updates
- **Message Delivery** - Configure delivery receipt settings

### MMS Settings
- **MMS Support** - Enable multimedia messaging service
- **Media URL** - Configure media file handling for MMS
- **File Size Limits** - Set maximum file size for media attachments
- **Supported Formats** - Configure supported media file formats
- **Content Filtering** - Enable content filtering for incoming media

## Phone Number Management

### Number Configuration

- **Geographic Numbers** - Local phone numbers by area code

### Number Assignment
