# Plivo Credentials

Configure your Plivo account credentials and communication platform settings for voice, SMS, and messaging services.

## Account Configuration

### Authentication Credentials
- **Auth ID** - Your Plivo Account Authentication ID
- **Auth Token** - Authentication token for API access
- **API Base URL** - Plivo API endpoint URL (default or custom)
- **Subaccount Auth ID** - Configure subaccounts for organizational separation
- **Subaccount Auth Token** - Authentication token for subaccount access

### Account Settings
- **Account Name** - Descriptive name for your Plivo account
- **Account Type** - Development, production, or trial account
- **Region Configuration** - Select Plivo region for optimal performance
- **Timezone** - Configure account timezone for scheduling and reporting
- **Currency** - Set preferred currency for billing and pricing

## Voice Services Configuration

### Voice Settings
- **Answer URL** - Primary webhook URL for incoming voice calls
- **Answer Method** - HTTP method (GET/POST) for voice webhooks
- **Hangup URL** - Webhook URL for call completion events
- **Fallback URL** - Backup URL for voice call handling
- **Machine Detection** - Enable answering machine detection

### Call Features
- **Caller ID** - Configure outbound caller ID presentation
- **Call Recording** - Enable automatic call recording capabilities
- **Call Screening** - Set up call screening and filtering options
- **Conference Calling** - Configure conference room settings
- **Call Queuing** - Set up call queue and hold music

### International Calling
- **Country Permissions** - Enable calling to specific countries
- **Rate Limits** - Set spending limits for international calls
- **Fraud Detection** - Enable fraud detection for international calls
- **Call Quality** - Configure call quality preferences by region
- **Compliance** - Configure country-specific compliance settings

## SMS and Messaging

### SMS Configuration
- **Message URL** - Webhook URL for incoming SMS messages
- **Message Method** - HTTP method for SMS webhooks
- **Delivery URL** - URL for SMS delivery status callbacks
- **Character Encoding** - Configure character encoding for SMS
- **Long Message Handling** - Split or reject long messages

### MMS Settings
- **MMS Support** - Enable multimedia messaging capabilities
- **Media Handling** - Configure media file processing
- **File Size Limits** - Set maximum file size for MMS attachments
- **Content Types** - Specify supported media content types
- **Delivery Receipts** - Enable MMS delivery confirmation

### Message Templates

- **Personalization** - Set up message personalization options
- **Compliance Templates** - Templates for regulatory compliance
- **A2P Messaging** - Application-to-Person messaging configuration
