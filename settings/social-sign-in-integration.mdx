**Interface Details:**
- **Location:** Displays the internal unique code or name for Go HighLevel (LeadConnector) locations.
- **Agent Name Field:** Lets you select or assign an agent to handle processes related to this location. If no agent is assignable, shows "No Fields Available".
- **Default:** Use the switch to mark a location as default.
- **Sync Contact:** Set contact sync scope (e.g., All, Answered, or No sync).
- **Assign User ID:** Toggle Yes/No to enable or disable user ID assignment for synced contacts.
- **Action:** Instantly "Re-Connect" to refresh integration, or "Disconnect" to remove the connection.

<Callout type="info">
To add a new LeadConnector integration, click <b>Connect New</b>. For embedding this integration table or management panel in another platform, use the <b>Embedding Link</b> provided at the top-right of the interface.
</Callout>

**Additional Controls:**
- <b>Showtest View:</b> Preview your current LeadConnector integrations.
- <b>Embedding Link:</b> Generate a special URL to embed the integration management table into external tools or dashboards.
