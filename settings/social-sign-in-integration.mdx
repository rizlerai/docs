Easily manage and configure your LeadConnector (Go HighLevel) locations and agent assignments.

- **Location:** Name or title of the Go HighLevel (LeadConnector) location.
- **Agent Name Field:** Select or assign the agent name (or "Not Public Available" if restricted).
- **Default:** Mark as the default location using a switch or toggle.
- **Sync Contact:** Choose the scope to sync contacts (e.g., All, Answered).
- **Assign User ID:** Toggle to enable or disable user ID assignment.
- **Action:** Instantly "Re-Connect" or "Disconnect" each location integration.
